从控制台中读取输入并存入变量中，有几个变量读几组输入

```shell
read <options> <variables>
```

| 参数   | 说明                              |
| ---- | ------------------------------- |
| `-p` | 显示提示信息                          |
| `-t` | 设置输入超时                          |
| `-n` | 读入 n 个字符，而不是一整行                 |
| `-r` | 读入原字符，即不将 `\` 理解为转义字符，而是 `\` 本身 |
| `-s` | 静默模式，不显示输入的内容，可用于密码等场景          |
