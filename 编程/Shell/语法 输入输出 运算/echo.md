使用 `echo` 输出后面的字符串，`-e` 可以识别转义字符

![[../../../../_resources/images/Pasted image 20241116013008.png]]

转义字符中，`\e` 或 `\033` 可以设置终端属性，如字体颜色、背景、光标等
- 以 `[` 开始，以 `m` 结束，以 `;` 分隔
- 后接 1 或 2 个数字，第一个数字表示样式，第二个表示颜色

| 第一位参数 | 格式                 |
| ---------- | -------------------- |
| `0`        | 无格式，后面不接数字 |
| `1`        | 加粗，后面不接数字   |
| `3`        | 前景色               |
| `4`        | 背景色               |

| 第二位参数 | 颜色   |
| ---------- | ------ |
| `0`        | 黑色   |
| `1`        | 红色   |
| `2`        | 绿色   |
| `3`        | 黄色   |
| `4`        | 蓝色   |
| `5`        | 紫色   |
| `6`        | 浅蓝色 |
| `7`        | 灰色   |
